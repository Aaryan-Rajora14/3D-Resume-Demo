## ✨ Project Overview

**3D Resume** is a powerful tool that lets you generate stunning, modern, responsive resume/portfolio websites — just like the one in the example.

The generated resumes feature:
- Sleek cyber-tech / dark mode design with neon accents
- Smooth tabbed navigation (About, Skills, Experience, Projects, Contact)
- Animated skill bars, floating profile photo, glassmorphic cards
- Fully responsive (mobile + desktop)
- Built-in timeline, project showcases, and contact section
- Zero dependencies — pure HTML + CSS + JS

## 🚀 Features

- **Modern UI/UX** — Orbitron + Syne typography, gradient accents, subtle grid background
- **Interactive Tabs** — Smooth switching between sections
- **Customizable Sections**:
  - Hero with photo & stats
  - Skills with animated progress bars
  - Experience & Education Timeline
  - Featured Projects + Project Table
  - Contact Information
- **One-File Output** — Easy to host anywhere (GitHub Pages, Netlify, Vercel, etc.)
- **Data-Driven** — Generate from JSON/YAML input (planned)
- **Export Options** — HTML, PDF (future), Print-optimized

---

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (custom properties + modern layout), Vanilla JavaScript
- **Styling**: Tailwind-inspired custom CSS with glassmorphism and cyber aesthetic
- **Generation Engine**: Python / Node.js (planned)
- **Templates**: Modular, extensible HTML templates
